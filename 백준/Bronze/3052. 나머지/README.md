### Problem Statement

<p>Given two natural numbers A and B, A%B is the remainder when A is divided by B. For example, the remainders of 7, 14, 27, and 38 when divided by 3 are 1, 2, 0, and 2.</p>

### Input

 <p>The first through tenth lines contain one number per line. These numbers are non-negative integers less than or equal to 1000.</p>

### Output

 <p>On the first line, print out the number of different remainders when divided by 42
