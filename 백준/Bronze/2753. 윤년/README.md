### Problem Description

<p>Given a year, write a program that prints 1 if the year is a leap year, and 0 otherwise.</p>

<p>A year is a leap year if both of the following conditions are satisfied: Itis divisible by 4, and not divisible by 100, OR It is divisible by 400.</p>

<p>For example: 2012 is a leap year because it is divisible by 4 but not by 100.</p>

### Input

 <p>The first line contains a year. The year is a natural number between 1 and 4000, inclusive.</p>

### Output 

 <p>Print 1 if the given year is a leap year; otherwise, print 0.</p>
