### Problem Description

<p>Junwon went to Costco for the first time last week. It was amazing.
However, despite buying only a few items, the total amount seemed suspiciously high!
Junwon decides to check the receipt to make sure the calculation was done correctly.</p>

<p>Given</p>

<ul>
	<li>The price and quantity of each purchased item</li>
	<li>The total amount written on the receipt</li>
</ul>

<p>The total amount written on the receipt.</p>

### Input

 <p>The first line contains the total amount X written on the receipt.</p>
 <p>The second line contains the number of different types of items N.</p>
 <p>The next N lines each contain two integers:</p>
 <ul>
	<li>a: the price of the item</li>
	<li>b: the quantity of the item</li>
</ul>
<p>Each pair is given with a space in between.</p>
 
### Output

 <p>Print <code>Yes</code> if the calculated total amount from the items matches the total amount on the receipt. Otherwise, print <code>No</code>.</p>
