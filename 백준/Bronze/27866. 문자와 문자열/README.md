### Problem Description
<p>Given a wors Ss and an integer Ii, write a program that prints the Iith letter of Ss.</p>

### Input
<p>The first line contains the word Ss, consisting of only lowercase and uppercase English letters. The maximum word length is 1000</p>
<p>The second line contains the integer Ii (1 ≤ i ≤ |S|. </p>

### Output
<p>Print the Iith letter of Ss.</p>
