### Problem Statement
<p>Professor M teaches a programming class at University X. There are 30 students in the class, and each student is assigned an attendance number from 1 to 30 on the student roster.</p>

<p>Twent-eight students submitted a special assignment assigned by the professor. Write a program to retrieve the attendance numbers of the two students who did not submit the assigment.</p>


### Input

<p>The input consists of 28 lines, each containing the attendance number n (1 ≤ n ≤ 30)of each submitter (student). There are no duplicate attendance numbers.</p>

### Output
<p>The output consists of two lines. The first line contains the smallest attendance number among the students who did not submit, and the second line contains the next lowest attendance number.</p>
