### Problem Description

<p>Given lower case for alphabet word. This time we need to check this word to palindrome.</p>

<p>Palindrome is the same word when read forward and backward. </p>

<p>level, noon is palindrome, baekjoon, online, judge is not palindrome.</p>

### Input

 <p>The first line given word. The word length is biggest 1 and smaller than 100, and only contain alphabet lower case.</p>

### Output

 <p>if palindrome then print 1 else 0.</p>
