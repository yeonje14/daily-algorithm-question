# [Bronze II] Number of words - 1152 

[Problem Link](https://www.acmicpc.net/problem/1152) 

### Performance Summary

Memory: 1972 KB, Time: 12 ms

### Categories

Implementation, String


### Problem Description

<p>You are given a string consisting of uppercase and lowercase English letters and spaces. How many words are there in this string? Write a program to find them. Note that if a word appears multiple times, you must count all occurrences.</p>

### Input

 <p>The first line contains a string consisting of uppercase and lowercase English letters and space. The string's length must not exceed 1,000,000. Words are separated by a single space, and there can be no consecutive spaces. The string can also begin or end with a space.</p>

### Output
 <p>The first line contains the number of words. </p>
