# [Bronze V] Compare the two integers - 1330 

[Problem Link](https://www.acmicpc.net/problem/1330) 

### Performance Summary

Memory: 17712 KB, Time: 172 ms

### Categories

Implementation

### Problem Description

<p>Given two integers A and B, write a program that compares A and B.</p>


### Input

 <p>The first line contains A and B, separated by a single space.</p>

### Output

 <p>On the first line, print one of the following three result:.</p>

<ul>
	<li>If A is greater than B, print '<code>></code>'.</li>
	<li>If A is less than B, print '<code><</code>'.</li>
	<li>If A and B are equal, print '<code>==</code>'.</li>
</ul>
