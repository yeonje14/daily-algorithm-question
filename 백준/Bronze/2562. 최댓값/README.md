### Problem Description

<p>Given nine distinct natural numbers, write a program that finds the maximum among them and determines the position of that maximum.</p>

<p>For example, given nine distinct natural numbers:</p>

<p>3, 29, 38, 12, 57, 74, 40, 85, 61</p>

<p>The maximum of these numbers is 85, which is the eighth number.</p>

### Input

<p>The first through ninth lines contain one natural number each. The given natural number is less than 100.</p>

### Output

<p>Print the maximum value on the first line, and the position of that maximum value on the second line.</p>
